Solution: bases de Python#

Une solution possible pour chaque exercice. Plusieurs formulations sont souvent valides — l’important est que la cellule produise le résultat attendu.

Variables et types#

# TODO : Créer une variable `temperature` (float) valant 18.5
# et une variable `ville` (str) valant "Lyon".

temperature = 18.5
ville = "Lyon"
print(temperature, ville)

18.5 Lyon

# TODO : Afficher "Il fait <temperature>°C à <ville>" avec une f-string.

print(f"Il fait {temperature}°C à {ville}")

Il fait 18.5°C à Lyon

# TODO : Convertir `temperature` en entier (cast) et l'afficher.
# Quelle valeur obtient-on ? Pourquoi ?

print(int(temperature))  # 18 — int() tronque vers zéro (pas d'arrondi)

18

# TODO : Créer une variable `mesure_manquante = None`.
# Afficher son type avec type().

mesure_manquante = None
print(type(mesure_manquante))  # <class 'NoneType'>

<class 'NoneType'>

Chaînes de caractères#

# TODO : Soit la chaîne phrase = "  Bonjour le monde  ".
# Supprimer les espaces autour (méthode .strip()) et la passer en minuscules.

phrase = "  Bonjour le monde  "
print(phrase.strip().lower())

bonjour le monde

# TODO : À partir de email = "jean.dupont@exemple.fr", récupérer le domaine
# ("exemple.fr") en utilisant .split("@").

email = "jean.dupont@exemple.fr"
print(email.split("@")[1])

exemple.fr

# TODO : Compter combien de fois la lettre "e" apparaît dans la chaîne
# "l'apprentissage automatique est partout".

texte = "l'apprentissage automatique est partout"
print(texte.count("e"))

4

Listes#

# TODO : Créer une liste `fruits` contenant : "pomme", "banane", "cerise".
# Afficher le premier et le dernier élément.

fruits = ["pomme", "banane", "cerise"]
print(fruits[0], fruits[-1])

pomme cerise

# TODO : Ajouter "datte" à la fin de la liste, puis afficher sa longueur.

fruits.append("datte")
print(len(fruits))

4

# TODO : Récupérer une sous-liste avec les deux premiers éléments (slicing).

print(fruits[:2])

['pomme', 'banane']

# TODO : Inverser la liste (méthode .reverse() ou slicing [::-1]) et l'afficher.

print(fruits[::-1])  # version slicing (n'altère pas la liste)
# fruits.reverse() ; print(fruits)  # version méthode (in-place)

['datte', 'cerise', 'banane', 'pomme']

Dictionnaires#

# TODO : Créer un dictionnaire `personne` avec les clés :
#   - "nom"   : "Martin"
#   - "age"   : 32
#   - "ville" : "Bordeaux"
# Afficher uniquement la valeur de "age".

personne = {"nom": "Martin", "age": 32, "ville": "Bordeaux"}
print(personne["age"])

32

# TODO : Ajouter la clé "email" avec la valeur "martin@exemple.fr".
# Modifier la valeur de "age" pour la passer à 33.

personne["email"] = "martin@exemple.fr"
personne["age"] = 33
print(personne)

{'nom': 'Martin', 'age': 33, 'ville': 'Bordeaux', 'email': 'martin@exemple.fr'}

# TODO : Tenter d'accéder à la clé "telephone" qui n'existe pas, avec
# une valeur par défaut "inconnu" (méthode .get()).

print(personne.get("telephone", "inconnu"))

inconnu

# TODO : Afficher toutes les clés, puis toutes les valeurs du dictionnaire
# (méthodes .keys() et .values()).

print(list(personne.keys()))
print(list(personne.values()))

['nom', 'age', 'ville', 'email']
['Martin', 33, 'Bordeaux', 'martin@exemple.fr']

Conditions#

# TODO : Écrire un bloc if/elif/else qui, à partir d'une variable
# `note` (entre 0 et 20), affiche :
#   - "insuffisant"  si note < 10
#   - "passable"     si 10 <= note < 14
#   - "bien"         si 14 <= note < 17
#   - "très bien"    sinon
# Tester avec note = 12.

note = 12
if note < 10:
    print("insuffisant")
elif note < 14:
    print("passable")
elif note < 17:
    print("bien")
else:
    print("très bien")

passable
