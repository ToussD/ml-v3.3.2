Solution: Pandas - Manipuler et analyser vos données#

Cet exercice utilisera le dataset mtcars qui comprend 32 observations sur 11 variables :

  1. mpg Miles/(US) gallon

  2. cyl Number of cylinders

  3. disp Displacement (cu.in.)

  4. hp Gross horsepower

  5. drat Rear axle ratio

  6. wt Weight (1000 lbs)

  7. qsec 1/4 mile time

  8. vs Engine (0 = V-shaped, 1 = straight)

  9. am Transmission (0 = automatic, 1 = manual)

  10. gear Number of forward gears

  11. carb Number of carburators

Dataframes et Indexes#

Les lignes d’un dataframe sont toujours indexées afin de permettre de les référencer. Par défaut, l’index d’un dataframe est une suite consécutive d’entier : 0, 1, 2, …

Exemples :

df.iloc[1:3,:]  # Retourne un dataframe référencant les ligne 1 à 2 et toutes les colonnes

df.loc[3:7,'col1':'col5'] # Retourne un dataframe référencant les ligne 3 à 7 et les colonnes 'col1' à 'col5' (par nom de colonne)

Apply#

La fonction pandas.Series.apply est très souvent utilisée. Elle permet d’appliquer une fonction à l’ensemble des valeurs d’une colonne d’un dataframe (qui est de type Serie) : Cette fonction peut-être standard (i.e. fournie par Python), ou d’une fonction que vous aurez implémentée selon vos besoin.

  • La fonction apply retourne une Serie (i.e. une colonne) qui est souvent affectée à une nouvelle colonne du dataframe d’origine.

  • Elle prend une fonction thefunc en paramètre

  • thefunc sera appelée autant de fois qu’il y a de lignes dans la Seriesur laquelle on l’applique

  • le paramétre de thefunc correspond à la valeur de l’élément en cours de traitement dans la Serieen cours

Exemple 1 :

def nb_mots(s):
    return len(s.split(' '))

dataframe['nb_word'] = dataframe['sentence'].apply(nb_mots)

Exemple 2 :

En déclarant une lambda expression (i.e. une fonction anonyme) :

dataframe['nb_word'] = dataframe['sentence'].apply(lambda s: len(s.split(' ')))
